# Project Overview 
  This project aims to provide an in-depth analysis of the Android app market by leveraging publicly available Google Play Store data. Through data cleaning, 
  extensive exploratory data analysis (EDA), visualization, and user sentiment analysis, the study investigates key market dynamics, app distribution trends, 
  and user perception. <br>
